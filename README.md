### Lab66 Object: Calculator1

- read() ให้รับค่า input 2 ค่า
- sum() ให้ return ค่าผลบวกของเลขทั้งสอง
- mul() ให้ return ค่าผลคูณของเลขทั้งสอง

```JavaScript
let calculator = {
  // ... your code ...
};

calculator.read();
alert(calculator.sum());
alert(calculator.mul());
```
